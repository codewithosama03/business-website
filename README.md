## Adex Business Website – Responsive Business Landing Page

##About

Adex Business Website is a responsive business landing page designed to showcase services, projects, and modern UI layout patterns.

This project focuses on structured layout design, responsive behavior, hero sections, service presentation, and clean UI styling using pure HTML and CSS.

It was built as a frontend UI project to practice real-world business website layouts commonly used for agencies, startups, and corporate portfolios.

The project emphasizes clean structure, reusable CSS variables, smooth transitions, and responsiveness across desktop and mobile devices.

This project will continue to evolve with UI improvements and additional sections as part of frontend development practice.

## Features

Responsive navigation bar

Sticky header with animation

Modern hero section with slider

Services showcase section

About section with accordion

Features and stats sections

Project showcase cards

Call-to-action (CTA) section

Footer with newsletter input

Hover effects and UI transitions

Mobile responsive design

Well-structured and reusable CSS

Project Structure / Components

Navbar: Responsive navigation bar with mobile menu

Hero Section: Landing banner with slider and call-to-action

Service Section: Business services cards

About Section: Accordion-style content layout

Features Section: Highlighted features with icons

Stats Section: Business statistics display

Project Section: Portfolio/project showcase

CTA Section: Call-to-action for user engagement

Footer: Contact info, links, and newsletter form

Responsive CSS: Media queries for all screen sizes
